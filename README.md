# remember

스마일게이트 개발 캠프 2023 - Remember

![image](./resources/harmony.png)

## 팀 소개
- 저희 팀은 총 백엔드 4명으로 이루어진 팀입니다. 
- 각각 Spring 개발자 2명, Node.js 개발자 1명, ASP.NET Core 개발자 1명으로 이루어져 있습니다.

![image](./resources/팀원소개.jpg)
